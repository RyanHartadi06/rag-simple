# Panduan Wawancara Kerja

Riset perusahaan dan job description. Siapkan contoh pengalaman (STAR: Situation, Task, Action, Result). Siapkan pertanyaan untuk pewawancara. Saat wawancara: jawab terstruktur, contoh konkret, jujur. Ajukan pertanyaan tentang peran dan tim. Email thank-you dalam 24 jam. Follow-up jika perlu. Ringkas: persiapan, STAR, pertanyaan, thank-you, profesional.
