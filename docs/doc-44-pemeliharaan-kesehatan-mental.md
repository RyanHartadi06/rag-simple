# Pemeliharaan Kesehatan Mental di Tempat Kerja

Kenali tanda: mood berubah lama, menarik diri, sulit fokus, tidur atau nafsu makan berubah, kelelahan. Batasan: jam kerja, ketersediaan; komunikasi ke atasan; hindari overcommit. Rutinitas: tidur cukup, aktivitas fisik, istirahat, batas work-life. Coping sehat: napas, bicara, menulis; hindari coping merugikan. Dukungan profesional: konseling, EAP jika ada. Di tim: normalisasi wellbeing, kebijakan cuti, leader model batasan. Ringkas: kenali tanda, batasan, rutinitas, coping, dukungan, budaya.
