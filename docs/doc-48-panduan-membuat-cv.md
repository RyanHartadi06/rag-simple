# Panduan Membuat CV

Satu halaman ideal untuk pemula; dua halaman OK untuk pengalaman banyak. Format jelas: nama, kontak, ringkasan singkat, pengalaman (tahun, peran, perusahaan, bullet pencapaian), pendidikan, skill, sertifikasi. Gunakan kata kerja dan angka (meningkatkan X%, memimpin Y). Sesuaikan dengan job description: sorot kata kunci relevan. Proofread; konsisten font dan spacing. Simpan PDF; nama file profesional (Nama_CV.pdf). Ringkas: singkat, terukur, relevan, rapi.
