# Kubernetes - Dasar

Kubernetes mengatur container: deployment, scaling, networking. Pod = unit terkecil (satu atau lebih container). Deployment mengelola replicas dan rolling update. Service memberi endpoint stabil untuk Pod (ClusterIP, NodePort, LoadBalancer). ConfigMap dan Secret untuk config. Namespace untuk isolasi. Ingress untuk HTTP routing. Helm untuk packaging chart. Ringkas: Pod, Deployment, Service, ConfigMap/Secret, Namespace, Ingress, Helm.
