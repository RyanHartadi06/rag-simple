# Agile dan Scrum - Overview

Agile: iteratif, fokus nilai dan feedback. Scrum: Product Owner (backlog, prioritas), Scrum Master (fasilitasi), Development Team. Artifact: Product Backlog, Sprint Backlog, Increment. Event: Sprint, Planning, Daily Scrum, Review, Retrospective. Transparansi dan adaptasi. Ringkas: PO, SM, Team; backlog dan Sprint; Planning, Daily, Review, Retro.
