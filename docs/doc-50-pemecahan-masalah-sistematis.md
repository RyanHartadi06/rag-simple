# Pemecahan Masalah Sistematis

Definisikan masalah: apa yang salah, untuk siapa, dalam kondisi apa. Kumpulkan fakta: observasi, data, batasan. Generate opsi: brainstorm tanpa kritik dulu; daftar solusi mungkin. Evaluasi: kriteria (biaya, waktu, risiko); bandingkan opsi. Pilih dan rencanakan: keputusan, langkah, siapa melakukan apa. Implementasi dan monitor: jalankan; ukur hasil; sesuaikan jika perlu. Refleksi: apa yang dipelajari; dokumentasi untuk masalah serupa. Alat: 5 Why, fishbone diagram, SWOT. Ringkas: definisi, fakta, opsi, evaluasi, rencana, eksekusi, refleksi.
