# Pemrograman Reactive - Konsep

Reactive: aliran data (stream) dan reaksi terhadap event. Observables/Streams: sumber yang mengeluarkan nilai over time; subscribe untuk consume. Operator: map, filter, merge, debounce, dll. Backpressure: konsumen mengontrol laju. Use case: UI event, HTTP response, WebSocket, state flow. Library: RxJS (JavaScript), Project Reactor (Java). Manfaat: komposisi async, cancellation, satu model untuk banyak sumber. Ringkas: stream, subscribe, operator, backpressure.
